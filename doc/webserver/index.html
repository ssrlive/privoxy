JunkBuster

JunkBuster is a web proxy which can block adverts, cookies, pop-up windows, and many other annoyances.

Please see the following links:

  • The project page on Sourceforge
  • Download recent releases
  • Mailing lists for support or to contribute
  • Frequently Asked Questions
  • User manual
  • Developer manual
  • The CVS repository can also be browsed online.

JunkBuster was originally written by JunkBusters Corporation, and was released as free open-source software under the GNU GPL. Stefan Waldherr made many improvements, before starting the SourceForge project to continue development. These sites still have some useful information, but not all of it is relevent to the latest builds.
